----Experiment 3----

In this experiment I decided to do the poertry project, I started this project using ai then I went on to add my own touches and changes, I started off with some simple prompts and I built from there adding litile details and changes here and there. I chose to do the story of the Insy Winsy Spider which is a childrens story because I felt like p5 could be used to make visuals for teaching in schools therefore I wanted to make an example of somthing like that. 

Once I was happy with how the poem / story appeared on the screen I thought that it would be good to add some visuals so I made some littile spiders dotted around the poem to make it more visually appealing. With the help of AI I also added this feature where when your mouse hovered over the screen anywhere littile words would pop up to do with the story as I thought that that add some engagment if this was shown in schools. 

Unfortunaly I forgot to make seperated iterations as I was very focused on making this the best I could but I learnt a lot of what p5 has to offer and I hope to get to use it in my future projects
